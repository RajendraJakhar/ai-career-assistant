# 🤖 AI Career Decision Assistant

## 🚀 Overview
This project is an AI-powered assistant that helps users choose the right career path based on their interests, skill level, time availability, and budget.

It uses a multi-step decision process and generates a complete career roadmap using LLM-based reasoning.

---

## 🧠 Features
- Multi-step conversation flow
- Option-based input system
- AI-powered career recommendations
- Generates:
  - Career Path
  - Top Courses
  - Learning Roadmap
  - Daily Study Plan
- Personalized reasoning based on user input

---

## 🛠 Tech Stack
- Flowise (Visual AI Builder)
- OpenRouter API
- LLM (GPT / Mixtral)

---

## 🔄 How It Works
The assistant asks questions step-by-step:

Goal → Interest → Level → Time → Budget → AI Output

After collecting all inputs, it generates a structured and personalized career plan.
